# Horse racing duals

### [Link](https://www.codingame.com/ide/puzzle/horse-racing-duals)

The Casablanca Hippodrome is organizing a new type of horse race: duels. In a duel, only two horses participate in the race. For the race to be interesting, two horses with similar power must be selected.

Write a program that, given a set of powers, identifies the two closest power values and outputs their difference as a positive integer.

### Input
Line 1: The number N of horses

The next N lines: The power Pi of each horse. Pi is an integer.

### Output  
The difference D between the two closest power values. D is a positive integer.
