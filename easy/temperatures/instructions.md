# Températures

### [Link](https://www.codingame.com/ide/puzzle/temperatures)

**Rules:**    
Write a program that displays the temperature closest to 0 among the input data. If two numbers are equally close to zero, then the positive integer will be considered closer to zero (for example, if the temperatures are -5 and 5, then display 5).

### Input  
Line 1: The number N of temperatures to analyze.

Line 2: A string containing the N temperatures expressed as integers ranging from -273 to 5526.

### Output  
Display 0 (zero) if no temperature is provided. Otherwise, display the temperature closest to 0.

### Constraints  
0 ≤ N < 10000
